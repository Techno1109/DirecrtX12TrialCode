DirecrtX12TrialCode
====

## Overview  
DirectX12でPMDとVMDを読み込んで描画します

## Description
PMDとVMDを読み込んで描画する他に、レイマーチングや深度を用いた輪郭抽出等が含まれています。  
ライセンスの都合等あるため、このままでは足りないライブラリがあります。  
そのため、このままでは実行できません。  
実行物というよりは実装の参考例としてみてもらえればと思います
